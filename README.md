# PySide6_Serial_Tools

#### 介绍
基于PySide6开发的串口工具。为学习PySide6初期的练手项目。

#### 软件架构
MV架构

#### 安装教程

```
pip install -r requirements.txt
```

#### 使用说明

1. exe运行  
运行./build文件夹下的exe文件

2. 源代码运行
进入虚拟环境，安装requirements.txt后运行main.py
